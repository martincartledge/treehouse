# Practicing Express


*nodemon*

`npm install -g nodemon`

*node-inspector*

`npm install -g node-inspector`

##### Run debugger and nodemon simultaneously

`node-inspector`

*new tab*

`nodemon --debug src/app.js`

*break on first line*

`nodemon --debug-brk src/app.js`

*install jade*

`npm install jade --save`
